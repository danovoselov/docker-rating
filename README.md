# docker-rating

## Project setup
```
git clone https://github.com/dimaaxxy/docker-rating.git
```
```angular2html
cd docker-rating/
```
```
docker-compose up -d
```
```
cd frontend/
```
```
npm install
```
```
npm run serve
```

After going to [localhost](http://localhost:8081/)
